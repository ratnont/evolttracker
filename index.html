Setup Instructions

  1. Go to console.cloud.google.com → APIs & Services → Credentials
  2. Create an OAuth 2.0 Client ID (type: Web application)
  3. Add your site URL to Authorized JavaScript origins
    (e.g. http://localhost:8080 or your hosted URL)
  4. Copy the Client ID above
  5. Enable the Google Drive API in your project
  6. Create a Drive folder, upload your Evolt PDFs, then paste the folder ID above
    (ID is the last part of the folder URL)
  7. Click Connect & Sync — a Google sign-in window will appear
